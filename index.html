Forks I keep coming back to

A few of the forked projects that best show the kinds of problems I like working on.

I want to modernize and revive a few oldies but goodies among the community and canonical plugins for WordPress. Authorship explores a well-architected approach to multi-author attribution. Comment Popularity is a simple method for letting users rate comments and using those ratings predictively to automate which comments get promoted or moderated. Agentic tooling is now part of my workflows, so I have borrowed and built on others’ agent-skills, Claude WordPress skills, general skills, and AI-assisted docs for writing and maintaining technical docs, plus broader WordPress and AI-assisted development workflows. You’ll also see some learning-oriented projects in my forked repos for workshops with the Edmonton WordPress Meetup, including Headless WordPress SvelteKit Site.
